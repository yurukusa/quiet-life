# quiet life

Touch the dark. Something alive appears.

**[▶ Play](https://yurukusa.github.io/quiet-life/)** · [itch.io](https://yurukusa.itch.io/quiet-life)

An interactive meditation. Touch the darkness and watch unique creatures appear — pulsing lights, orbital dancers, ribbon swimmers, spiral growers, gentle wanderers, and blooming flowers. Each one is born with its own DNA.

- Space or double-tap for auto mode
- Single HTML file, no dependencies
- No tracking, no ads, no login

## Healing series

- **quiet life** — Touch to create beings
- [deep breath](https://github.com/yurukusa/deep-breath) — Breathing meditation with generative particles
- [star moss](https://github.com/yurukusa/star-moss) — Drag to grow vines that bloom into stars
